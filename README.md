**HCOWA Brief Helper** 是一款专门为西非健康共同体协会打造的轻量化桌面内容生产工具。

---

## ✨ 核心特性

- **云端同步**: 采用 `https://sousuo.zze.cc` 实时引擎，无需本地环境搭建。
- **现代化 UI**: 深度契合 Windows 11 设计风格，支持卡片式内容编辑。
- **一键触达**: 在软件内完成生成、即时编辑校对、并一键推送到 Telegram 指定群组。
- **免安装运行**: 提供编译好的 `.exe` 文件，双击即用。

---

## 🛠️ 如何使用

1. **下载安装**: 访问 [Releases](https://github.com/cshaizhihao/hcowa-brief-automation/releases) 下载最新的绿色版 `HCOWA_Brief_Helper.zip`。
2. **生成数据**: 点击 **“☁️ 同步云端数据”**，软件将自动抓取最新的西非及加纳医疗时事、股市行情、并套用规定模板。
3. **人工校对**: 抓取的数据会显示在中间的编辑框中，您可以手动调整细节（如更新招商电话或股市数字）。
4. **一键推送**: 检查无误后，点击 **“🚀 确认正式推送”**。

## 🏗️ 开发者指南 (如果您需要自行编译)

如果您需要从源码运行或修改逻辑：

```bash
# 安装依赖
pip install -r requirements.txt

# 运行程序
python main.py

# 打包为单个 EXE (Windows)
pip install pyinstaller
pyinstaller --noconsole --onefile --icon=app.ico main.py
```
